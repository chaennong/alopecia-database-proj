# AlopeciaDatabaseProj

## Overview
Synthetic clinical dataset and reproducible analysis workflow comparing JAK inhibitor outcomes. Demographics, treatments, and outcomes are simulated to preserve the real-data structure.

## Repository Structure
- 'data/': synthetic dataset
- 'R/': data simulation scripts
- 'visualization/': visualization of synthetic data
- 'analysis/' R markdown analysis report

## Disclaimer
All data are fully synthetic and do not contain any real patient information.
